# Atlas des Inégalités

Comprendre comment les inégalités façonnent nos sociétés.

Atlas des Inégalités est un projet pédagogique visant à rendre accessibles les connaissances scientifiques sur les liens entre les inégalités sociales et les grands indicateurs de société :

- Santé
- Éducation
- Logement
- Criminalité
- Mobilité sociale
- Confiance sociale
- Démocratie

L'objectif est d'aider chacun à comprendre les données, les mécanismes et les débats scientifiques liés aux inégalités.

Le projet s'adresse aux citoyens, aux lycéens, aux étudiants, aux enseignants, aux journalistes et aux décideurs publics.

## Principes

- S'appuyer sur des données vérifiables
- Présenter les résultats de la recherche
- Expliquer les mécanismes en jeu
- Présenter les limites et les débats scientifiques
- Rester accessible au plus grand nombre

## Vision

Explorer les données.
Comprendre les mécanismes.
Éclairer le débat.
